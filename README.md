# Kaggle Dataset Evaluator Agent

An AI-powered dataset evaluation system that helps determine the quality, usability, and machine learning suitability of datasets.

## Features

- Dataset Overview
- Missing Value Analysis
- Duplicate Detection
- Feature Analysis
- Identifier Detection
- ML Task Inference
- Dataset Scoring
- Project Recommendations
- MCP Server Integration

## Architecture

Dataset
↓
Analysis Engine
↓
MCP Tools
↓
MCP Server

## Tech Stack

- Python
- Pandas
- MCP
- Google ADK
- Gemini
- Streamlit (planned)
